# 🎵 Spotify Top 50 Songs Analytics Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard built using Spotify Top 50 Global data. The dashboard explores song performance, artist trends, popularity patterns, album types, and release trends to uncover insights into global music consumption.

The goal of this project was to transform raw music data into meaningful insights through interactive visualizations and analytical storytelling.

---

## 📊 Key Insights

* Singles dominated the Top 50 charts more frequently than albums.
* Recent releases accounted for a significant share of top-performing songs.
* A small group of artists contributed a large share of chart entries.
* Higher popularity scores were generally associated with stronger chart performance.
